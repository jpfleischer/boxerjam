This page requires Pale Moon + the Flash NPAPI plugin. Flash is end-of-life and unsafe on the open web. Use a dedicated browser profile or a VM, and only enable Flash for this site.
Setup steps:
1. Install Pale Moon.
2. Install a Flash NPAPI plugin you trust. Adobe discontinued Flash, so use an archived installer only if you can verify its source and checksum.
3. In Pale Moon, enable the Flash plugin and allow it for this site.
